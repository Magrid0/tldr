# mate-screenshot

> Crea screenshot nell'ambiente desktop MATE.
> Maggiori informazioni: <https://manned.org/mate-screenshot>.

- Crea uno screenshot a tutto schermo:

`mate-screenshot`

- Crea uno screenshot della finestra attiva:

`mate-screenshot --window`

- Crea uno screenshot di un'area specifica:

`mate-screenshot --area`

- Crea uno screenshot in modo interattivo:

`mate-screenshot --interactive`

- Crea uno screenshot senza bordi:

`mate-screenshot --window --remove-border`

- Crea uno screenshot con un effetto specifico:

`mate-screenshot --effect={{shadow|border|none}}`

- Crea uno screenshot con un ritardo specifico in secondi:

`mate-screenshot --delay={{5}}`
