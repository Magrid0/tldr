# pkgctl

> Frontend unificato per i devtools di Arch Linux.
> Maggiori informazioni: <https://manned.org/pkgctl>.

- Consulta la documentazione per autenticare `pkgctl` con servizi come GitLab:

`tldr pkgctl auth`

- Consulta la documentazione per compilare pacchetti in una `chroot` pulita:

`tldr pkgctl build`

- Consulta la documentazione per aggiornare il repository binario come passo finale di rilascio:

`tldr pkgctl db update`

- Consulta la documentazione per confrontare file di pacchetti usando diverse modalità:

`tldr pkgctl diff`

- Consulta la documentazione per rilasciare artefatti di compilazione:

`tldr pkgctl release`

- Consulta la documentazione per gestire repository Git di pacchetti e la loro configurazione:

`tldr pkgctl repo`

- Mostra versione:

`pkgctl version`
