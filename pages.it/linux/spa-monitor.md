# spa-monitor

> Carica un plugin SPA e ne istanzia un dispositivo.
> Nota: Questa utilità è principalmente destinata al debugging dei plugin dei dispositivi.
> Maggiori informazioni: <https://docs.pipewire.org/page_man_spa-monitor_1.html>.

- Carica un plugin SPA specifico:

`spa-monitor {{percorso/del/plugin.so}}`
