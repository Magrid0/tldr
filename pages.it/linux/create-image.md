# create-image

> Crea immagini CPIO.
> Maggiori informazioni: <https://git.kernel.org/pub/scm/bluetooth/bluez.git/tree/tools/create-image.c>.

- Crea un archivio CPIO dalla directory corrente:

`create-image {{[-o|--output]}} {{path/to/output.cpio}}`

- Mostra aiuto:

`create-image {{[-h|--help]}}`
