# fwupdtool

> Aggiorna manualmente il firmware dei dispositivi o manipola i file del firmware.
> Vedi anche: `fwupdmgr`.
> Maggiori informazioni: <https://github.com/fwupd/fwupd/blob/main/src/fwupdtool.md>.

- Mostra tutti i dispositivi rilevati da `fwupd`:

`fwupdtool get-devices`

- Installa il firmware da un file:

`fwupdtool install {{path/to/firmware}}`

- Mostra aiuto:

`fwupdtool {{[-h|--help]}}`
