# xdg-user-dir

> Recupera le posizioni delle directory utente XDG.
> Vedi anche: `xdg-user-dirs-update`.
> Maggiori informazioni: <https://manned.org/xdg-user-dir>.

- Mostra la home directory per l'utente corrente:

`xdg-user-dir`

- Mostra la posizione di una directory utente:

`xdg-user-dir {{DESKTOP|DOWNLOAD|TEMPLATES|PUBLICSHARE|DOCUMENTS|MUSIC|PICTURES|VIDEOS|...}}`
