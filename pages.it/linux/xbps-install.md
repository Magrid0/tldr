# xbps-install

> Utilità XBPS per (re)installare e aggiornare pacchetti.
> Vedi anche: `xbps`.
> Maggiori informazioni: <https://manned.org/xbps-install>.

- Installa un nuovo pacchetto:

`xbps-install {{pacchetto}}`

- Sincronizza e aggiorna tutti i pacchetti:

`xbps-install {{[-Su|--sync --update]}}`
