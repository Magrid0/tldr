# latte-dock

> Dock sostitutivo per il desktop Plasma.
> Maggiori informazioni: <https://github.com/KDE/latte-dock>.

- Pulisce la cache QML:

`latte-dock --clear-cache`

- Importa e carica il layout predefinito all'avvio:

`latte-dock --default-layout`

- Carica un layout specifico all'avvio:

`latte-dock --layout {{layout_name}}`

- Importa e carica un layout specifico:

`latte-dock --import-layout {{path/to/file}}`
