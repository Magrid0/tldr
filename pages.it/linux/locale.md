# locale

> Ottiene informazioni specifiche sulla locale.
> Maggiori informazioni: <https://manned.org/locale>.

- Elenca tutte le variabili d'ambiente globali che descrivono la locale dell'utente:

`locale`

- Elenca tutte le locali disponibili:

`locale {{[-a|--all-locales]}}`

- Mostra tutte le locali disponibili e i metadati associati:

`locale {{[-a|--all-locales]}} {{[-v|--verbose]}}`

- Mostra il formato della data corrente:

`locale date_fmt`
