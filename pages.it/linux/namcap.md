# namcap

> Controlla pacchetti binari e `PKGBUILD` sorgente per errori comuni di impacchettamento.
> Maggiori informazioni: <https://manned.org/namcap>.

- Controlla uno specifico file `PKGBUILD`:

`namcap {{percorso/del/pkgbuild}}`

- Controlla uno specifico file pacchetto:

`namcap {{percorso/del/pacchetto.pkg.tar.zst}}`

- Controlla un file, mostrando messaggi informativi aggiuntivi:

`namcap {{[-i|--info]}} {{percorso/del/file}}`
