# darling

> Esegue software macOS su Linux.
> Maggiori informazioni: <https://docs.darlinghq.org/darling-shell.html>.

- Esegue un comando integrato:

`darling shell {{uname}}`

- Esegue un programma specifico con argomenti:

`darling shell {{path/to/program}} {{program_argument_1 program_argument_2 ...}}`

- Apre una shell macOS:

`darling shell`

- Arresta il servizio:

`darling shutdown`
