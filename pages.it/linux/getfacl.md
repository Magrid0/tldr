# getfacl

> Ottiene le liste di controllo degli accessi (ACL) dei file.
> Maggiori informazioni: <https://manned.org/getfacl>.

- Mostra la lista di controllo degli accessi del file:

`getfacl {{path/to/file_or_directory}}`

- Mostra la lista di controllo degli accessi del file con ID numerici di utente e gruppo:

`getfacl {{[-n|--numeric]}} {{path/to/file_or_directory}}`

- Mostra la lista di controllo degli accessi del file in formato tabellare:

`getfacl {{[-t|--tabular]}} {{path/to/file_or_directory}}`
