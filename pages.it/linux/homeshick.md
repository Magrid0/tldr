# homeshick

> Sincronizza i dotfile Git.
> Vedi anche: `chezmoi`, `stow`, `tuckr`, `vcsh`.
> Maggiori informazioni: <https://github.com/andsens/homeshick/wiki>.

- Crea un nuovo castello:

`homeshick generate {{castle_name}}`

- Aggiungi un file al tuo castello:

`homeshick track {{castle_name}} {{path/to/file}}`

- Vai a un castello:

`homeshick cd {{castle_name}}`

- Clona un castello:

`homeshick clone {{github_username}}/{{repository_name}}`

- Crea collegamenti simbolici a tutti i file di un castello:

`homeshick link {{castle_name}}`
