# pkgctl db update

> Aggiorna il database di `pacman` come passo finale di rilascio per pacchetti che sono stati trasferiti e organizzati su <https://repos.archlinux.org>.
> Maggiori informazioni: <https://manned.org/pkgctl-db-update>.

- Aggiorna il repository binario come passo finale di rilascio:

`pkgctl db update`
