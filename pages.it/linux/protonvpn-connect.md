# protonvpn connect

> Connetti a Proton VPN.
> Maggiori informazioni: <https://protonvpn.com/support/linux-cli#how-to-use-the-cli-for-linux>.

- Connetti a Proton VPN usando il server più veloce disponibile:

`protonvpn connect`

- Connetti al server più veloce in un paese specifico:

`protonvpn connect --country "{{codice_o_nome_paese}}"`

- Connetti al server più veloce in una città specifica:

`protonvpn connect --city "{{nome_città}}"`

- Connetti a un server specifico:

`protonvpn connect {{nome_server}}`

- Mostra aiuto:

`protonvpn connect {{[-h|--help]}}`
