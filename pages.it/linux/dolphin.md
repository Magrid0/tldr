# dolphin

> Il gestore di file di KDE per gestire file e directory.
> Vedi anche: `nautilus`, `caja`, `thunar`, `ranger`.
> Maggiori informazioni: <https://apps.kde.org/dolphin/>.

- Avvia il gestore di file:

`dolphin`

- Apre directory specifiche:

`dolphin {{path/to/directory1 path/to/directory2 ...}}`

- Apre con file o directory specifici selezionati:

`dolphin --select {{path/to/file_or_directory1 path/to/file_or_directory2 ...}}`

- Apre una nuova finestra:

`dolphin --new-window`

- Apre directory specifiche in vista divisa:

`dolphin --split {{path/to/directory1}} {{path/to/directory2}}`

- Avvia il demone (richiesto solo per usare l'interfaccia D-Bus):

`dolphin --daemon`

- Mostra aiuto:

`dolphin {{[-h|--help]}}`
