# talk

> Programma di comunicazione visiva che copia le righe dal tuo terminale a quello di un altro utente.
> Maggiori informazioni: <https://www.gnu.org/software/inetutils/manual/inetutils.html#talk-invocation>.

- Avvia una sessione talk con un utente sulla stessa macchina:

`talk {{nome_utente}}`

- Avvia una sessione talk con un utente sulla stessa macchina, connesso su tty3:

`talk {{nome_utente}} {{tty3}}`

- Avvia una sessione talk con un utente su una macchina remota:

`talk {{nome_utente}}@{{hostname}}`

- Pulisci il testo su entrambi gli schermi dei terminali:

`<Ctrl d>`

- Esci dalla sessione talk:

`<Ctrl c>`
