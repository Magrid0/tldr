# systemctl default

> Entra nella modalità predefinita del sistema.
> Maggiori informazioni: <https://www.freedesktop.org/software/systemd/man/latest/systemctl.html#default>.

- Entra in modalità predefinita come operazione bloccante:

`systemctl default`

- Entra in modalità predefinita come operazione asincrona:

`systemctl default --no-block`
