# fancontrol

> Gestione automatica della velocità delle ventole.
> Utilizza un file di configurazione per controllare un'ampia gamma di variabili.
> Maggiori informazioni: <https://manned.org/fancontrol>.

- Avvia `fancontrol` utilizzando il file di configurazione predefinito `/etc/fancontrol`:

`fancontrol`

- Avvia `fancontrol` utilizzando un percorso personalizzato per il file di configurazione:

`fancontrol {{path/to/config_file}}`
