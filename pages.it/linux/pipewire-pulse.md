# pipewire-pulse

> Avvia un demone compatibile con PulseAudio per PipeWire.
> Maggiori informazioni: <https://docs.pipewire.org/page_man_pipewire-pulse_1.html>.

- Avvia il demone con impostazioni predefinite:

`pipewire-pulse`

- Avvia il demone con un file di configurazione specifico:

`pipewire-pulse {{[-c|--config]}} {{percorso/pipewire-pulse.conf}}`

- Aumenta il livello di verbosità (usa più volte per maggiore verbosità, es. `-vvv`):

`pipewire-pulse {{[-v|--verbose]}}`

- Mostra aiuto:

`pipewire-pulse {{[-h|--help]}}`
