# qm help

> Mostra l'aiuto per un comando.
> Maggiori informazioni: <https://pve.proxmox.com/pve-docs/qm.1.html#cli_qm_help>.

- Mostra l'aiuto per un comando specifico:

`qm {{[h|help]}} {{command}}`

- Mostra l'aiuto per un comando specifico con informazioni dettagliate:

`qm {{[h|help]}} {{command}} --verbose {{true|false}}`
