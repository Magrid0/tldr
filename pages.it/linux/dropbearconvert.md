# dropbearconvert

> Converte tra formati di chiave privata Dropbear e OpenSSH.
> Maggiori informazioni: <https://manned.org/dropbearconvert>.

- Converte una chiave privata OpenSSH in formato Dropbear:

`dropbearconvert openssh dropbear {{path/to/input_key}} {{path/to/output_key}}`

- Converte una chiave privata Dropbear in formato OpenSSH:

`dropbearconvert dropbear openssh {{path/to/input_key}} {{path/to/output_key}}`
