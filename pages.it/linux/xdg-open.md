# xdg-open

> Apre un file o URL nell'applicazione preferita dell'utente.
> Maggiori informazioni: <https://portland.freedesktop.org/doc/xdg-open.html>.

- Apri la directory corrente nel file explorer predefinito:

`xdg-open .`

- Apri un URL nel browser predefinito:

`xdg-open {{https://example.com}}`

- Apri un'immagine nel visualizzatore di immagini predefinito:

`xdg-open {{path/to/image}}`

- Apri un PDF nel visualizzatore PDF predefinito:

`xdg-open {{path/to/pdf}}`

- Mostra aiuto:

`xdg-open --help`
