# Terraria

> Crea e avvia un server Terraria senza interfaccia grafica.
> Maggiori informazioni: <https://terraria.wiki.gg/wiki/Server>.

- Avvia una configurazione interattiva del server:

`{{percorso/di}}/TerrariaServer`

- Avvia un server Terraria:

`{{percorso/di}}/TerrariaServer -world {{percorso/del/mondo.wld}}`
