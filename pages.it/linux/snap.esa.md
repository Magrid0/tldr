# snap

> Sentinel Application Platform (SNAP) per elaborare dati satellitari dell'Agenzia Spaziale Europea (ESA).
> Maggiori informazioni: <https://step.esa.int/main/download/snap-download/>.

- Mostra tutti gli aggiornamenti:

`snap --nosplash --nogui --modules --list --refresh`

- Mostra aiuto:

`snap --help`
