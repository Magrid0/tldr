# setfont

> Imposta il font del terminale per i TTY.
> Maggiori informazioni: <https://manned.org/setfont>.

- Cambia il font del terminale:

`setfont {{font}}.gz`

- Cambia il font del terminale specificando un percorso:

`setfont /usr/share/kbd/{{mapping}}/{{font.ext}}`

- Raddoppia la dimensione del font:

`setfont {{[-d|--double]}}`

- Ripristina il font predefinito:

`setfont`
