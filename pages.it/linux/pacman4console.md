# pacman4console

> Un gioco testuale per console ispirato al Pacman originale.
> Maggiori informazioni: <https://github.com/YoctoForBeaglebone/pacman4console>.

- Avvia una partita al Livello 1:

`pacman4console`

- Avvia una partita a un certo livello (ci sono nove livelli ufficiali):

`pacman4console --level={{numero_livello}}`

- Avvia l'editor di livelli di pacman4console, salvando in un file di testo specificato:

`pacman4consoleedit {{percorso/del/file_livello}}`

- Gioca un livello personalizzato:

`pacman4console --level={{percorso/del/file_livello}}`
