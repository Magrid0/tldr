# wpexec

> Esegue script Lua di WirePlumber.
> Vedi anche: `wpctl`, `wireplumber`.
> Maggiori informazioni: <https://pipewire.pages.freedesktop.org/wireplumber/scripting/lua_api/lua_introduction.html>.

- Esegue uno script WirePlumber:

`wpexec {{percorso/del/file.lua}}`

- Mostra aiuto:

`wpexec --help`
