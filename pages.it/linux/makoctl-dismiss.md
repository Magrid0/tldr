# makoctl dismiss

> Ignora le notifiche in `mako`.
> Maggiori informazioni: <https://manned.org/makoctl>.

- Ignora la notifica più recente:

`makoctl dismiss`

- Ignora una notifica specifica tramite ID:

`makoctl dismiss -n {{notification_id}}`

- Ignora tutte le notifiche:

`makoctl dismiss {{[-a|--all]}}`

- Ignora tutte le notifiche nel primo gruppo:

`makoctl dismiss {{[-g|--group]}}`

- Ignora senza aggiungere alla cronologia:

`makoctl dismiss {{[-h|--no-history]}}`
