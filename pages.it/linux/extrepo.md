# extrepo

> Gestisce i repository Debian esterni.
> Viene utilizzato per gestire repository esterni in Debian.
> Maggiori informazioni: <https://manned.org/extrepo.1p>.

- Cerca un dato pacchetto:

`extrepo search {{pacchetto}}`

- Abilita il repository:

`sudo extrepo enable {{nome_repository}}`

- Disabilita il repository:

`sudo extrepo disable {{nome_repository}}`

- Aggiorna il repository:

`sudo extrepo update {{nome_repository}}`
