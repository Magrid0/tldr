# systemctl daemon-reload

> Ricarica la configurazione del gestore systemd.
> Usa questo comando dopo aver creato, modificato o eliminato file di unità.
> Vedi anche: `systemctl reload`.
> Maggiori informazioni: <https://www.freedesktop.org/software/systemd/man/latest/systemctl.html#daemon-reload>.

- Ricarica systemd per applicare le modifiche ai file di unità:

`systemctl daemon-reload`
