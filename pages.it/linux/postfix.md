# postfix

> Programma di controllo del mail transfer agent (MTA) Postfix.
> Maggiori informazioni: <https://www.postfix.org/postfix.1.html>.

- Controlla la configurazione:

`sudo postfix check`

- Controlla lo stato del demone Postfix:

`sudo postfix status`

- Avvia Postfix:

`sudo postfix start`

- Ferma gentilmente Postfix:

`sudo postfix stop`

- Svuota la coda di posta:

`sudo postfix flush`

- Ricarica i file di configurazione:

`sudo postfix reload`
