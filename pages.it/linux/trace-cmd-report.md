# trace-cmd report

> Mostra gli eventi di trace registrati.
> Vedi anche: `trace-cmd list`, `trace-cmd record`.
> Maggiori informazioni: <https://manned.org/trace-cmd-report>.

- Mostra il trace registrato:

`sudo trace-cmd report`

- Mostra il trace registrato per una CPU specifica:

`sudo trace-cmd report --cpu {{numero_cpu}}`
