# spa-json-dump

> Converte un file SPA JSON in JSON standard.
> Maggiori informazioni: <https://docs.pipewire.org/page_man_spa-json-dump_1.html>.

- Converte un file SPA JSON in JSON standard:

`spa-json-dump {{percorso/del/file.conf}}`

- Converte JSON SPA da `stdin` in JSON standard:

`echo "{{stringa_spa_json}}" | spa-json-dump`
