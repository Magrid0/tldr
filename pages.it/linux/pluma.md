# pluma

> Modifica file nell'ambiente desktop MATE.
> Maggiori informazioni: <https://manned.org/pluma>.

- Avvia l'editor:

`pluma`

- Apri documenti specifici:

`pluma {{percorso/file1 percorso/file2 ...}}`

- Apri documenti usando una codifica specifica:

`pluma --encoding {{WINDOWS-1252}} {{percorso/file1 percorso/file2 ...}}`

- Mostra tutte le codifiche supportate:

`pluma --list-encodings`

- Apri un documento e vai a una riga specifica:

`pluma +{{10}} {{percorso/del/file}}`
