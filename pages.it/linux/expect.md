# expect

> Esecutore di script che interagisce con altri programmi che richiedono input dall'utente.
> Maggiori informazioni: <https://manned.org/expect>.

- Esegue uno script expect da un file:

`expect {{path/to/file}}`

- Esegue uno script expect specificato:

`expect -c "{{comandi}}"`

- Entra in una REPL [i]nterattiva (usa `exit` o `<Ctrl d>` per uscire):

`expect -i`
