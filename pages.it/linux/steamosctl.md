# steamosctl

> Controlla il sistema SteamOS.
> Maggiori informazioni: <https://gitlab.com/evlaV/steamos-manager>.

- Passa alla modalità desktop:

`steamosctl switch-to-desktop-mode`

- Passa alla modalità gioco:

`steamosctl switch-to-game-mode`

- Cambia se il sistema apre la modalità gioco o desktop durante il login:

`steamosctl set-default-login-mode {{game|desktop}}`

- Mostra aiuto:

`steamosctl`
