# dphys-swapfile

> Gestisce il file di swap su sistemi Linux basati su Debian.
> Maggiori informazioni: <https://manned.org/dphys-swapfile>.

- Disabilita il file di swap:

`dphys-swapfile swapoff`

- Abilita il file di swap:

`dphys-swapfile swapon`

- Crea un nuovo file di swap:

`dphys-swapfile setup`
