# ydotool

> Controlla input di tastiera e mouse tramite comandi in modo agnostico dal server di visualizzazione.
> Maggiori informazioni: <https://github.com/ReimuNotMoe/ydotool#usage>.

- Avvia il demone ydotool in background:

`ydotoold`

- Esegui un input di `<LeftClick>`:

`ydotool click 0xC0`

- Esegui un input di `<RightClick>`:

`ydotool click 0xC1`

- Input `<Alt F4>`:

`ydotool key 56:1 62:1 62:0 56:0`
