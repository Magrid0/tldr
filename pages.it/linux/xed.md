# xed

> Modifica file nell'ambiente desktop Cinnamon.
> Maggiori informazioni: <https://manned.org/xed>.

- Avvia l'editor:

`xed`

- Apri file specifici:

`xed {{path/to/file1 path/to/file2 ...}}`

- Apri file usando una codifica specifica:

`xed --encoding {{WINDOWS-1252}} {{path/to/file1 path/to/file2 ...}}`

- Stampa tutte le codifiche supportate:

`xed --list-encodings`

- Apri un file e vai a una riga specifica:

`xed +{{10}} {{path/to/file}}`
