# ubuntu-security-status

> Mostra informazioni sul supporto di sicurezza per i pacchetti Ubuntu installati.
> Maggiori informazioni: <https://git.launchpad.net/ubuntu/+source/update-manager/tree/ubuntu-security-status>.

- Mostra il numero di pacchetti non supportati:

`ubuntu-security-status`

- Elenca i pacchetti non più disponibili per il download:

`ubuntu-security-status --unavailable`

- Elenca i pacchetti di terze parti:

`ubuntu-security-status --thirdparty`
