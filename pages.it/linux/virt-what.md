# virt-what

> Script shell che rileva se sei in esecuzione in una macchina virtuale.
> Emette un codice specifico per una macchina virtuale se rilevata.
> Maggiori informazioni: <https://manned.org/virt-what>.

- Rileva se sei in esecuzione in una macchina virtuale:

`virt-what`

- Registra l'output di `virt-what` in un file:

`virt-what > {{percorso/del/file}}`

- Mostra versione:

`virt-what --version`
