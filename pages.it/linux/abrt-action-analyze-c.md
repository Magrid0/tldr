# abrt-action-analyze-c

> Calcola l'UUID per una directory di dati di problema con `coredump`.
> Maggiori informazioni: <https://manned.org/abrt-action-analyze-c>.

- Calcola e salva l'UUID per la directory di lavoro corrente:

`abrt-action-analyze-c`

- Calcola e salva l'UUID per una directory specifica:

`abrt-action-analyze-c -d {{path/to/directory}}`

- Calcola e salva l'UUID in modo verboso:

`abrt-action-analyze-c -v`
