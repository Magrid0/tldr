# id3v2

> Gestisce i tag id3v2, converte e elenca id3v1.
> Maggiori informazioni: <https://manned.org/id3v2>.

- Elenca tutti i generi:

`id3v2 {{[-L|--list-genres]}}`

- Elenca tutti i tag di file specifici:

`id3v2 {{[-l|--list]}} {{path/to/file1 path/to/file2 ...}}`

- Elimina tutti i tag `id3v2` o `id3v1` di file specifici:

`id3v2 {{--delete-v2|--delete-v1}} {{path/to/file1 path/to/file2 ...}}`

- Mostra aiuto:

`id3v2 {{[-h|--help]}}`

- Mostra versione:

`id3v2 {{[-v|--version]}}`
