# xeyes

> Mostra occhi sullo schermo che seguono il cursore del mouse.
> Maggiori informazioni: <https://manned.org/xeyes>.

- Avvia xeyes sul display predefinito della macchina locale:

`xeyes`

- Avvia xeyes sul display 0, schermo 0 di una macchina remota:

`xeyes -display {{host_remoto}}:{{0}}.{{0}}`
