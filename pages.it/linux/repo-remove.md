# repo-remove

> Utilità di manutenzione del database dei pacchetti che rimuove pacchetti da un repository locale.
> Vedi anche: `repo-add`.
> Maggiori informazioni: <https://manned.org/repo-add>.

- Rimuove un pacchetto da un repository locale:

`repo-remove {{path/to/database.db.tar.gz}} {{package}}`
