# ausyscall

> Mappa nomi e numeri di syscall.
> Maggiori informazioni: <https://manned.org/ausyscall>.

- Mostra il numero di syscall di una chiamata di sistema specifica:

`ausyscall {{pattern_di_ricerca}}`

- Mostra il nome di un numero di syscall specifico:

`ausyscall {{numero_chiamata_sistema}}`

- Mostra tutte le syscall per un'architettura specifica:

`ausyscall {{architettura}} --dump`
