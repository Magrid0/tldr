# steamos-chroot

> Cambia la directory root in un ambiente SteamOS.
> Maggiori informazioni: <https://gitlab.com/users/evlaV/projects>.

- Passa all'altra partizione A/B:

`steamos-chroot {{[-p|--partset]}} other`

- Passa a una partizione su un altro disco:

`steamos-chroot {{[-d|--disk]}} {{/dev/sdX}} {{[-p|--partset]}} {{A|B}}`

- Mostra aiuto:

`steamos-chroot {{[-h|--help]}}`
