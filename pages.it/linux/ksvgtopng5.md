# ksvgtopng5

> Converte file SVG in formato PNG.
> Maggiori informazioni: <https://invent.kde.org/plasma/kde-cli-tools/-/blob/master/ksvgtopng/ksvgtopng.cpp>.

- Converte un file SVG (deve essere un percorso assoluto) in PNG:

`ksvgtopng5 {{width}} {{height}} {{path/to/file.svg}} {{output_filename.png}}`
