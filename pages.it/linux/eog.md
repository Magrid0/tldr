# eog

> Visualizza e gestisce immagini.
> Maggiori informazioni: <https://manned.org/eog>.

- Apre un file immagine:

`eog {{path/to/immagine.png}}`

- Apre più immagini:

`eog {{immagine1.jpg immagine2.png ...}}`

- Apre immagini in modalità a schermo intero:

`eog --fullscreen {{path/to/immagine.jpg}}`

- Avvia una presentazione:

`eog --slide-show {{path/to/directory}}`

- Apre una singola immagine senza raccolta di immagini:

`eog --single-window {{path/to/immagine.jpg}}`

- Apre un'immagine in una nuova istanza di finestra:

`eog --new-instance {{path/to/immagine.jpg}}`
