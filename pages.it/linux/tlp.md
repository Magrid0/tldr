# tlp

> Gestione avanzata dell'alimentazione per Linux.
> Vedi anche: `tlp-stat`.
> Maggiori informazioni: <https://linrunner.de/tlp/usage/tlp.html>.

- Applica le impostazioni (in base alla fonte di alimentazione effettiva):

`sudo tlp start`

- Applica le impostazioni della batteria (ignorando la fonte di alimentazione effettiva):

`sudo tlp bat`

- Applica le impostazioni AC (ignorando la fonte di alimentazione effettiva):

`sudo tlp ac`
