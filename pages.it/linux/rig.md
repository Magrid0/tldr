# rig

> Utilità per comporre un nome casuale, cognome, numero civico e indirizzo, insieme a città, stato, CAP e prefisso geograficamente coerenti (cioè corrispondono tutti alla stessa area).
> Maggiori informazioni: <https://manned.org/rig>.

- Mostra un nome casuale (maschile o femminile) e indirizzo:

`rig`

- Mostra un nome casuale [m]aschile (o [f]emminile) e indirizzo:

`rig -{{m|f}}`

- Usa file [d]ati da una directory specifica (default è `/usr/share/rig`):

`rig -d {{path/to/directory}}`

- Mostra un numero specifico di identità:

`rig -c {{number}}`

- Mostra un numero specifico di identità [f]emminili:

`rig -f -c {{number}}`
