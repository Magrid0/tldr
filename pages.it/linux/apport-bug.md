# apport-bug

> Invia una segnalazione di bug su Ubuntu.
> Maggiori informazioni: <https://wiki.ubuntu.com/Apport>.

- Segnala un bug riguardante l'intero sistema:

`apport-bug`

- Segnala un bug riguardante un pacchetto specifico:

`apport-bug {{pacchetto}}`

- Segnala un bug riguardante un eseguibile specifico:

`apport-bug {{path/to/eseguibile}}`

- Segnala un bug riguardante un processo specifico:

`apport-bug {{PID}}`
