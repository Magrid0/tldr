# mssh

> Client SSH basato su GTK+ per interagire con più server SSH contemporaneamente.
> Maggiori informazioni: <https://manned.org/mssh>.

- Apre una nuova finestra e si connette a più server SSH:

`mssh {{user@host1 user@host2 ...}}`

- Apre una nuova finestra e si connette a un gruppo di server predefiniti in `~/.mssh_clusters`:

`mssh {{[-a|--alias]}} {{alias_name}}`
