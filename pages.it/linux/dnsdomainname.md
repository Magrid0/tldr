# dnsdomainname

> Mostra il nome di dominio DNS del sistema.
> Nota: Lo strumento usa `gethostname` per ottenere l'hostname del sistema e poi `getaddrinfo` per risolverlo in un nome canonico.
> Maggiori informazioni: <https://www.gnu.org/software/inetutils/manual/inetutils.html#dnsdomainname-invocation>.

- Mostra il nome di dominio DNS del sistema:

`dnsdomainname`
