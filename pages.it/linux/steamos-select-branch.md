# steamos-select-branch

> Seleziona un ramo di rilascio del sistema SteamOS.
> Nota: questo comando è deprecato, usa `atomupd-manager` invece.
> Maggiori informazioni: <https://gitlab.com/users/evlaV/projects>.

- Visualizza il ramo corrente:

`steamos-select-branch -c`

- Elenca tutti i rami disponibili:

`steamos-select-branch -l`

- Passa al ramo beta:

`steamos-select-branch beta`

- Torna al rilascio stabile:

`steamos-select-branch stable`

- Passa al ramo all'avanguardia:

`steamos-select-branch main`
