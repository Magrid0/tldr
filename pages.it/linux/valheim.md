# Valheim

> Crea e avvia un server Valheim senza interfaccia grafica.
> Nota: Configura il server modificando una copia dello script `start_server.sh`.
> Maggiori informazioni: <https://www.valheimgame.com/support/a-guide-to-dedicated-servers/>.

- Avvia il server:

`{{percorso/di}}/start_server.sh`

- Ferma il server correttamente:

`<Ctrl c>`

- Avvia il server usando lo script Docker fornito:

`{{percorso/di/docker_start_server.sh}} {{percorso/di/start_server.sh}}`
