# xdg-terminal-exec

> Apre il terminale predefinito.
> Maggiori informazioni: <https://manned.org/xdg-terminal-exec>.

- Apri il terminale predefinito:

`xdg-terminal-exec`

- Esegui un comando nel terminale predefinito:

`xdg-terminal-exec {{comando}} {{argomento1 argomento2 ...}}`

- Tieni il terminale aperto dopo l'esecuzione del comando:

`xdg-terminal-exec --hold {{comando}}`
