# mandb

> Gestisce il database delle pagine di manuale pre-formattate.
> Maggiori informazioni: <https://manned.org/mandb>.

- Pulisce ed elabora le pagine di manuale:

`mandb`

- Aggiorna una singola voce:

`mandb --filename {{path/to/file}}`

- Crea voci da zero invece di aggiornare:

`mandb --create`

- Elabora solo i database utente:

`mandb --user-db`

- Non rimuovere voci obsolete:

`mandb --no-purge`

- Verifica la validità delle pagine di manuale:

`mandb --test`
