# prt-get

> Il gestore pacchetti CRUX.
> Maggiori informazioni: <https://crux.nu/doc/prt-get%20-%20User%20Manual.html>.

- Installa un pacchetto:

`prt-get install {{pacchetto}}`

- Installa un pacchetto con gestione delle dipendenze:

`prt-get depinst {{pacchetto}}`

- Aggiorna un pacchetto manualmente:

`prt-get upgrade {{pacchetto}}`

- Rimuovi un pacchetto:

`prt-get remove {{pacchetto}}`

- Aggiorna il sistema dall'albero delle porte locale:

`prt-get sysup`

- Cerca nell'albero delle porte:

`prt-get search {{query}}`

- Cerca un file in un pacchetto:

`prt-get fsearch {{file}}`
