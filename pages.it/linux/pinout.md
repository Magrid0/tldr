# pinout

> Visualizza le informazioni sui pin GPIO del Raspberry Pi corrente sul terminale con un diagramma ASCII.
> Maggiori informazioni: <https://www.raspberrypi.com/documentation/computers/raspberry-pi.html#view-a-gpio-pinout-for-your-raspberry-pi>.

- Visualizza le informazioni sui pin e il diagramma dell'intestazione GPIO per il Raspberry Pi corrente:

`pinout`

- Apre <https://pinout.xyz/> nel browser predefinito:

`pinout {{[-x|--xyz]}}`
