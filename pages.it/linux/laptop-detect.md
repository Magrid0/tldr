# laptop-detect

> Tenta di determinare se lo script è in esecuzione su un laptop o desktop.
> Maggiori informazioni: <https://gitlab.com/debiants/laptop-detect>.

- Restituisce uno stato di uscita 0 se il dispositivo è probabilmente un laptop, altrimenti restituisce 1:

`laptop-detect`

- Stampa il tipo di dispositivo rilevato per il sistema corrente:

`laptop-detect --verbose`

- Mostra versione:

`laptop-detect --version`
