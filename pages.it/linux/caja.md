# caja

> Gestisce file e directory nell'ambiente desktop MATE.
> Vedi anche: `nautilus`, `dolphin`, `thunar`, `ranger`.
> Maggiori informazioni: <https://manned.org/caja>.

- Apre la home directory dell'utente corrente:

`caja`

- Apre directory specifiche in finestre separate:

`caja {{path/to/directory1 path/to/directory2 ...}}`

- Apre directory specifiche in schede:

`caja {{[-t|--tabs]}} {{path/to/directory1 path/to/directory2 ...}}`

- Apre una directory con una dimensione specifica della finestra:

`caja {{[-g|--geometry]}} {{600}}x{{400}} {{path/to/directory}}`

- Chiude tutte le finestre:

`caja {{[-q|--quit]}}`
