# pipewire

> Avvia il demone PipeWire.
> Maggiori informazioni: <https://docs.pipewire.org/page_man_pipewire_1.html>.

- Avvia il demone PipeWire:

`pipewire`

- Usa un file di configurazione diverso:

`pipewire --config {{percorso/del/file.conf}}`

- Imposta il livello di verbosità (error, warn, info, debug o trace):

`pipewire -{{v|vv|...|vvvvv}}`

- Mostra aiuto:

`pipewire --help`
