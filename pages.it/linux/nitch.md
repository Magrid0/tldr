# nitch

> Un piccolo e incredibilmente veloce system fetch scritto interamente in Nim.
> Maggiori informazioni: <https://github.com/ssleert/nitch#usage->.

- Mostra le informazioni di sistema (hostname, kernel, uptime, ecc.):

`nitch`

- Mostra aiuto:

`nitch {{[-h|--help]}}`

- Mostra versione:

`nitch {{[-v|--version]}}`
