# megatools-dl

> Scarica file da `mega.nz`.
> Parte della suite `megatools`.
> Maggiori informazioni: <https://xff.cz/megatools/man/megatools-dl.html>.

- Scarica file da un link `mega.nz` nella directory corrente:

`megatools-dl {{https://mega.nz/...}}`

- Scarica file da un link `mega.nz` in una directory specifica:

`megatools-dl --path {{path/to/directory}} {{https://mega.nz/...}}`

- Scegli interattivamente quali file scaricare:

`megatools-dl --choose-files {{https://mega.nz/...}}`

- Limita la velocità di download in KiB/s:

`megatools-dl --limit-speed {{speed}} {{https://mega.nz/...}}`
