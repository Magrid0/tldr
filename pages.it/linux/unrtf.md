# unrtf

> Converte file RTF in altri formati di file.
> Maggiori informazioni: <https://manned.org/unrtf>.

- Converte un file RTF in HTML (default):

`unrtf {{percorso/del/file.rtf}}`

- Converte un file RTF in ASCII semplice:

`unrtf --text {{percorso/del/file.rtf}}`

- Converte un file RTF in LaTeX:

`unrtf --latex {{percorso/del/file.rtf}}`
