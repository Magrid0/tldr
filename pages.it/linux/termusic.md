# termusic

> Lettore musicale da terminale scritto in Rust che utilizza tasti di scelta rapida simili a vim.
> Vedi anche: `cmus`, `ncmpcpp`, `audacious`.
> Maggiori informazioni: <https://github.com/tramhao/termusic>.

- Apri termusic in una directory specifica. (Può essere impostata permanentemente in `~/.config/termusic/config.toml`):

`termusic {{percorso/della/directory}}`

- Disabilita la visualizzazione della copertina dell'album per un file specifico:

`termusic -c {{percorso/del/file_musicale}}`

- Mostra aiuto:

`termusic --help`
