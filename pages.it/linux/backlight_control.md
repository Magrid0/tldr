# backlight_control

> Controlla la retroilluminazione di un computer Linux usando valori percentuali.
> Maggiori informazioni: <https://github.com/Hendrikto/backlight_control#usage>.

- Aumenta/riduce la retroilluminazione di una specifica percentuale:

`backlight_control {{+|-}}{{5}}`

- Imposta l'intensità della retroilluminazione a una specifica percentuale:

`backlight_control {{90}}`

- Mostra aiuto:

`backlight_control`
