# hyprpm

> Controlla i plugin per il compositore Wayland Hyprland.
> Maggiori informazioni: <https://wiki.hypr.land/Plugins/Using-Plugins/#hyprpm>.

- Aggiunge un plugin:

`hyprpm add {{git_url}}`

- Rimuove un plugin:

`hyprpm remove {{git_url|plugin_name}}`

- Abilita un plugin:

`hyprpm enable {{plugin_name}}`

- Disabilita un plugin:

`hyprpm disable {{plugin_name}}`

- Aggiorna e controlla tutti i plugin:

`hyprpm update`

- Forza un'operazione:

`hyprpm {{[-f|--force]}} {{operation}}`

- Elenca tutti i plugin installati:

`hyprpm list`
