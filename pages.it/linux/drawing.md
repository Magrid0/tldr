# drawing

> Editor di immagini raster di base gratuito nell'ambiente desktop GNOME.
> Maggiori informazioni: <https://maoschanz.github.io/drawing/>.

- Avvia Drawing:

`drawing`

- Apre file specifici:

`drawing {{path/to/image1 path/to/image2 ...}}`

- Apre file specifici in una nuova finestra:

`drawing --new-window {{path/to/image1 path/to/image2 ...}}`
