# nitrogen

> Browser e impostatore di sfondi del desktop per X Window.
> Maggiori informazioni: <https://manned.org/nitrogen>.

- Visualizza e imposta gli sfondi da una directory specifica:

`nitrogen {{percorso/della/directory}}`

- Imposta lo sfondo con impostazioni di dimensione automatiche:

`nitrogen --set-auto {{percorso/del/file}}`

- Ripristina lo sfondo precedente:

`nitrogen --restore`
