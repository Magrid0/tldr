# i3status

> Barra di stato per il window manager i3.
> Questo comando viene solitamente chiamato dal file di configurazione di i3.
> Maggiori informazioni: <https://i3wm.org/i3status/manpage.html>.

- Stampa periodicamente la barra di stato su `stdout`, usando la configurazione predefinita:

`i3status`

- Stampa periodicamente la barra di stato su `stdout`, usando una configurazione specifica:

`i3status -c {{path/to/i3status.conf}}`

- Mostra aiuto e versione:

`i3status -h`
