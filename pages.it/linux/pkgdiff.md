# pkgdiff

> Confronta i contenuti dei file di due pacchetti Slackware.
> Maggiori informazioni: <https://slackware.nl/slackware/slackware64-current/source/a/pkgtools/manpages/pkgdiff.8>.

- Confronta due pacchetti e mostra le differenze:

`pkgdiff {{percorso/pacchetto1.txz}} {{percorso/pacchetto2.txz}}`

- Confronta due pacchetti con output colorato:

`pkgdiff -c {{percorso/pacchetto1.txz}} {{percorso/pacchetto2.txz}}`

- Confronta due pacchetti e mostra un output unificato semplice:

`pkgdiff -a {{percorso/pacchetto1.txz}} {{percorso/pacchetto2.txz}}`
