# autoexpect

> Genera uno script osservando una sessione.
> Maggiori informazioni: <https://manned.org/autoexpect>.

- Avvia una shell e genera uno script da essa:

`autoexpect`

- Esegue un comando e genera uno script da esso:

`autoexpect {{comando}}`

- Esegue un comando, genera uno script e lo salva in un file specificato:

`autoexpect -f {{path/to/file}} {{comando}}`
