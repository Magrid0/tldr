# toilet

> Mostra caratteri artistici ASCII.
> Maggiori informazioni: <http://caca.zoy.org/wiki/toilet>.

- Genera arte ASCII per un dato testo:

`toilet {{testo_input}}`

- Genera arte ASCII utilizzando un file di font personalizzato:

`toilet {{testo_input}} {{[-f|--font]}} {{nome_file_font}}`

- Genera arte ASCII utilizzando un filtro:

`toilet {{testo_input}} {{[-F|--filter]}} {{nome_filtro}}`

- Mostra i filtri toilet disponibili:

`toilet {{[-F|--filter]}} list`
