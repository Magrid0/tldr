# koji help

> Mostra informazioni di aiuto per i comandi Koji.
> Maggiori informazioni: <https://docs.pagure.org/koji/HOWTO/#exploring-koji>.

- Mostra tutti i comandi disponibili:

`koji help`

- Mostra aiuto per una categoria specifica (Le categorie disponibili sono: admin, all, bind, build, download, info, misc, monitor, search):

`koji help {{category}}`

- Mostra aiuto:

`koji help {{[-h|--help]}}`
