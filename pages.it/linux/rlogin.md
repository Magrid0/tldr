# rlogin

> Accede a un host remoto.
> Maggiori informazioni: <https://www.gnu.org/software/inetutils/manual/inetutils.html#rlogin-invocation>.

- Accede a un host remoto:

`rlogin {{remote_host}}`

- Accede a un host remoto con un nome utente specifico:

`rlogin {{[-l|--user]}} {{username}} {{remote_host}}`
