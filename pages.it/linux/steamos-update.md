# steamos-update

> Aggiorna SteamOS.
> Nota: questo comando è deprecato, usa `atomupd-manager` invece.
> Maggiori informazioni: <https://gitlab.com/users/evlaV/projects>.

- Aggiorna il sistema operativo:

`sudo steamos-update`

- Controlla se è disponibile un aggiornamento:

`sudo steamos-update check`
