# urxvt

> Rxvt-unicode.
> Un emulatore di terminale personalizzabile.
> Maggiori informazioni: <https://manned.org/urxvt>.

- Apri una nuova finestra urxvt:

`urxvt`

- Esegui in una directory specifica:

`urxvt -cd {{percorso/della/directory}}`

- Esegui un comando in una nuova finestra urxvt:

`urxvt -e {{comando}}`

- Esegui un comando e mantieni la finestra aperta:

`urxvt --hold -e {{comando}}`

- Esegui un comando all'interno della shell `sh`:

`urxvt -e {{sh}} -c {{comando}}`
