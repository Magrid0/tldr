# cacaclock

> Mostra l'ora corrente come arte ASCII.
> Maggiori informazioni: <https://packages.debian.org/sid/caca-utils>.

- Mostra l'ora:

`cacaclock`

- Cambia il font:

`cacaclock {{[-f|--font]}} {{font}}`

- Cambia il formato usando una specifica di formato `strftime`:

`cacaclock {{[-d|--dateformat]}} {{strftime_arguments}}`
