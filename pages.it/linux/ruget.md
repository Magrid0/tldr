# ruget

> Alternativa a wget scritta in Rust.
> Maggiori informazioni: <https://github.com/ksk001100/ruget>.

- Scarica il contenuto di un URL in un file:

`ruget {{https://example.com/file}}`

- Scarica il contenuto di un URL in un file di output specificato:

`ruget {{[-o|--output]}} {{file_name}} {{https://example.com/file}}`
