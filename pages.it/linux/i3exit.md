# i3exit

> Esce dal window manager i3.
> Maggiori informazioni: <https://gitlab.manjaro.org/packages/extra/i3exit/-/blob/master/i3exit>.

- Esce da i3:

`i3exit logout`

- Blocca i3:

`i3exit lock`

- Spegne il sistema:

`i3exit shutdown`

- Sospende il sistema:

`i3exit suspend`

- Passa alla schermata di login per accedere come un utente diverso:

`i3exit switch_user`

- Iberna il sistema:

`i3exit hibernate`

- Riavvia il sistema:

`i3exit reboot`
