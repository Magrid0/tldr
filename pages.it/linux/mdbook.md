# mdbook

> Crea libri online scrivendo file Markdown.
> Maggiori informazioni: <https://rust-lang.github.io/mdBook/cli/index.html>.

- Crea un progetto mdbook nella directory corrente:

`mdbook init`

- Crea un progetto mdbook in una directory specifica:

`mdbook init {{path/to/directory}}`

- Pulisce la directory con il libro generato:

`mdbook clean`

- Serve un libro su <http://localhost:3000>, compilazione automatica quando i file cambiano:

`mdbook serve`

- Osserva un insieme di file Markdown e compila automaticamente quando un file viene modificato:

`mdbook watch`
