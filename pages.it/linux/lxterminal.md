# lxterminal

> Emulatore di terminale per LXDE.
> Maggiori informazioni: <https://manned.org/lxterminal>.

- Apre una finestra di LXTerminal:

`lxterminal`

- Apre una finestra di LXTerminal, esegue un comando e poi esce:

`lxterminal {{[-e|--command]}} "{{command}}"`

- Apre una finestra di LXTerminal con più schede:

`lxterminal --tabs={{tab_name1,tab_name2,...}}`

- Apre una finestra di LXTerminal con un titolo specifico:

`lxterminal --title={{title_name}}`

- Apre una finestra di LXTerminal con una directory di lavoro specifica:

`lxterminal --working-directory={{path/to/directory}}`
