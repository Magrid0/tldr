# slabtop

> Mostra le informazioni sulla cache slab del kernel in tempo reale.
> Vedi anche: `top`, `htop`, `atop`.
> Maggiori informazioni: <https://manned.org/slabtop>.

- Avvia `slabtop`:

`sudo slabtop`

- Ordina per dimensione della [c]ache:

`sudo slabtop {{[-s|--sort]}} c`

- Ordina per numero di [o]ggetti:

`sudo slabtop {{[-s|--sort]}} o`

- Ordina per [d]imensione dell'oggetto:

`sudo slabtop {{[-s|--sort]}} s`

- Mostra una volta e poi esce:

`sudo slabtop {{[-o|--once]}}`

- Mostra aiuto:

`slabtop {{[-h|--help]}}`
