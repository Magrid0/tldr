# fincore

> Mostra quanta memoria cache sta occupando un file.
> Vedi anche: `fadvise`.
> Maggiori informazioni: <https://manned.org/fincore>.

- Mostra i dettagli della cache per un file:

`fincore {{path/to/file}}`

- Mostra tutte le colonne di dati possibili:

`fincore --output-all {{path/to/file}}`

- Mostra aiuto:

`fincore {{[-h|--help]}}`
