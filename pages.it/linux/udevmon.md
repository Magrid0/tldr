# udevmon

> Intercetta e monitora i dispositivi di input per avviare attività.
> Filtra o modifica gli eventi secondo i file di configurazione (default: `/etc/interception/udevmon.d/*.yaml`).
> Maggiori informazioni: <https://gitlab.com/interception/linux/tools/-/tree/master#udevmon>.

- Avvia udevmon con un file di configurazione specifico:

`udevmon -c {{percorso/del/config.yaml}}`
