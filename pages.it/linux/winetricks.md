# winetricks

> Gestisci ambienti Windows virtuali di Wine.
> Maggiori informazioni: <https://gitlab.winehq.org/wine/wine/-/wikis/Winetricks>.

- Avvia una configurazione grafica nella posizione predefinita di Wine:

`winetricks`

- Specifica una directory Wine personalizzata in cui eseguire Winetricks:

`WINEPREFIX={{percorso/della/directory_wine}} winetricks`

- Installa una DLL o un componente Windows nella directory Wine predefinita:

`winetricks {{pacchetto}}`
