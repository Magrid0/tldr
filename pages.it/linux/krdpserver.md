# krdpserver

> Avvia un server RDP.
> Maggiori informazioni: <https://invent.kde.org/plasma/krdp#running-the-example-server>.

- Avvia un server RDP con nome utente e password:

`krdpserver {{[-u|--username]}} {{user_name}} {{[-p|--password]}} {{password}}`

- Condivide un monitor specifico indicato dal suo indice:

`krdpserver {{[-u|--username]}} {{user_name}} {{[-p|--password]}} {{password}} --monitor {{n}}`
