# burpsuite

> Applicazione basata su GUI utilizzata principalmente nei test di penetrazione di applicazioni web.
> Maggiori informazioni: <https://portswigger.net/burp/documentation/desktop/troubleshooting/launch-from-command-line>.

- Avvia Burp Suite:

`burpsuite`

- Avvia Burp Suite usando la configurazione predefinita:

`burpsuite --use-defaults`

- Apre un file di progetto specifico:

`burpsuite --project-file={{path/to/file}}`

- Carica un file di configurazione specifico:

`burpsuite --config-file={{path/to/file}}`

- Avvia senza estensioni:

`burpsuite --disable-extensions`
