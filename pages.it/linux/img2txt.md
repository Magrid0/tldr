# img2txt

> Converte immagini in caratteri ASCII colorati e li outputta in file colorati basati su testo.
> Maggiori informazioni: <https://manned.org/img2txt>.

- Imposta il numero di colonne di output a un valore specifico:

`img2txt {{[-W|--width]}} {{10}} {{path/to/image}}`

- Imposta il numero di righe di output a un valore specifico:

`img2txt {{[-H|--height]}} {{5}} {{path/to/image}}`

- Imposta la larghezza del font di output a un valore specifico:

`img2txt {{[-x|--font-width]}} {{12}} {{path/to/image}}`

- Imposta l'altezza del font di output a un valore specifico:

`img2txt {{[-y|--font-height]}} {{14}} {{path/to/image}}`

- Imposta la luminosità dell'immagine a un valore specifico:

`img2txt {{[-b|--brightness]}} {{2}} {{path/to/image}}`
