# mpg123

> Lettore audio MPEG da console.
> Maggiori informazioni: <https://manned.org/mpg123>.

- Riproduce i file mp3 specificati:

`mpg123 {{path/to/file1.mp3 path/to/file2.mp3 ...}}`

- Riproduce l'mp3 da `stdin`:

`cat {{file.mp3}} | mpg123 -`

- Salta avanti alla prossima canzone:

`<f>`

- Torna all'inizio della canzone:

`<b>`

- Ferma o riavvia il file corrente:

`<s>`

- Avanzamento veloce:

`<.>`

- Esci:

`<q>`
