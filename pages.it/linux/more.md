# more

> Mostra interattivamente un file, permettendo lo scorrimento e la ricerca.
> Vedi anche: `less`.
> Maggiori informazioni: <https://manned.org/more>.

- Apre un file:

`more {{path/to/file}}`

- Mostra una riga specifica:

`more +{{line_number}} {{path/to/file}}`

- Vai alla pagina successiva:

`<Space>`

- Cerca una stringa (premi `<n>` per andare al risultato successivo):

`</>{{something}}<Enter>`

- Esci:

`<q>`

- Mostra aiuto sui comandi interattivi:

`<h>`
