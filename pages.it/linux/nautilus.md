# nautilus

> Esplora file predefinito per l'ambiente desktop GNOME.
> Conosciuto anche come GNOME Files.
> Vedi anche: `dolphin`, `caja`, `thunar`, `vifm`.
> Maggiori informazioni: <https://manned.org/nautilus>.

- Avvia Nautilus:

`nautilus`

- Avvia Nautilus come utente root:

`nautilus admin:/`

- Avvia Nautilus e mostra una directory specifica:

`nautilus {{percorso/della/directory}}`

- Avvia Nautilus con un file o una directory specifica selezionata:

`nautilus --select {{percorso/del/file_o_directory}}`

- Avvia Nautilus in una finestra separata:

`nautilus --new-window`

- Chiude tutte le istanze di Nautilus:

`nautilus --quit`

- Mostra aiuto:

`nautilus --help`
