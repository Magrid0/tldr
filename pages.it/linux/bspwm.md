# bspwm

> Window manager a mosaico basato sulla partizione binaria dello spazio.
> Vedi anche: `bspc`.
> Maggiori informazioni: <https://github.com/baskerville/bspwm/blob/master/doc/bspwm.1.asciidoc>.

- Avvia `bspwm` (nota: un window manager preesistente non deve essere aperto quando viene eseguito questo comando):

`bspwm -c {{path/to/config}}`
