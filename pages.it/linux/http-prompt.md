# http-prompt

> Un client HTTP interattivo con autocompletamento ed evidenziazione della sintassi.
> Maggiori informazioni: <https://github.com/httpie/http-prompt>.

- Avvia una sessione con l'URL predefinito di <http://localhost:8000> o la sessione precedente:

`http-prompt`

- Avvia una sessione con un dato URL:

`http-prompt {{http://example.com}}`

- Avvia una sessione con alcune opzioni iniziali:

`http-prompt {{localhost:8000/api}} --auth {{username:password}}`
