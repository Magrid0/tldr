# autorandr

> Cambia automaticamente il layout dello schermo.
> Maggiori informazioni: <https://manned.org/autorandr>.

- Salva il layout dello schermo corrente:

`autorandr {{[-s|--save]}} {{nome_profilo}}`

- Mostra i profili salvati:

`autorandr`

- Carica il primo profilo rilevato:

`autorandr {{[-c|--change]}}`

- Carica un profilo specifico:

`autorandr {{[-l|--load]}} {{nome_profilo}}`

- Imposta il profilo predefinito:

`autorandr {{[-d|--default]}} {{nome_profilo}}`
