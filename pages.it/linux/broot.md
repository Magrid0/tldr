# broot

> Esplora le directory in modo interattivo.
> Maggiori informazioni: <https://github.com/Canop/broot>.

- Installa o reinstalla la funzione shell `br`:

`broot --install`

- Avvia e naviga la directory corrente in modo interattivo:

`broot`

- Avvia mostrando la dimensione dei file e delle directory:

`broot {{[-s|--sizes]}}`

- Avvia ordinando per dimensione, mostrando file ignorati e nascosti:

`broot {{[-w|--whale-spotting]}}`

- Avvia mostrando i permessi:

`broot {{[-p|--permissions]}}`

- Avvia mostrando solo le directory:

`broot {{[-f|--only-folders]}}`

- Avvia mostrando file e directory nascosti:

`broot {{[-h|--hidden]}}`
