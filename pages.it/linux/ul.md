# ul

> Sottolinea un testo.
> Ogni carattere in una stringa deve essere sottolineato separatamente.
> Maggiori informazioni: <https://manned.org/ul>.

- Mostra il contenuto del file con sottolineature dove applicabile:

`ul {{file.txt}}`

- Mostra il contenuto del file con sottolineature fatte di trattini `-`:

`ul {{[-i|--indicated]}} {{file.txt}}`
