# kscreen-console

> Interroga lo stato di KScreen.
> Maggiori informazioni: <https://manned.org/kscreen-console>.

- Mostra tutte le uscite e i file di configurazione da allegare a una segnalazione di bug:

`kscreen-console bug`

- Mostra i percorsi dei file di configurazione di KScreen:

`kscreen-console config`

- Mostra le informazioni sulle uscite e la configurazione di KScreen:

`kscreen-console outputs`

- Monitora le modifiche:

`kscreen-console monitor`

- Mostra la configurazione corrente di KScreen come JSON:

`kscreen-console json`

- Mostra aiuto:

`kscreen-console {{[-h|--help]}}`

- Mostra aiuto incluse le opzioni della riga di comando specifiche di Qt:

`kscreen-console --help-all`
