# kinfocenter

> Centro informazioni KDE per mostrare informazioni di sistema.
> Maggiori informazioni: <https://manned.org/kinfocenter>.

- Apre l'interfaccia grafica:

`kinfocenter`

- Elenca tutti i moduli KCM disponibili per `kinfocenter`:

`kinfocenter --list`

- Mostra aiuto:

`kinfocenter {{[-h|--help]}}`
