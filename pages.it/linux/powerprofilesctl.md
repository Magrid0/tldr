# powerprofilesctl

> Rende disponibile la gestione dei profili di alimentazione tramite D-Bus.
> Maggiori informazioni: <https://gitlab.freedesktop.org/upower/power-profiles-daemon#how-to-use>.

- Elenca i profili di alimentazione disponibili:

`powerprofilesctl`

- Imposta un profilo di alimentazione specifico:

`powerprofilesctl set {{nome_profilo}}`
