# knockd

> Demone di port knocking per ascoltare i knock sulle porte ed eseguire script.
> Maggiori informazioni: <https://manned.org/knockd>.

- Avvia il demone di sistema knockd:

`knockd {{[-d|--daemon]}}`

- Usa un file di configurazione specifico per knockd:

`knockd {{[-c|--config]}} {{path/to/file}}.configuration`
