# mediamtx

> Server multimediale e proxy in tempo reale.
> Maggiori informazioni: <https://github.com/bluenviron/mediamtx>.

- Esegue MediaMTX:

`mediamtx`

- Esegue MediaMTX con una posizione di configurazione personalizzata:

`mediamtx {{path/to/config.yml}}`

- Avvia MediaMTX come demone:

`systemctl start mediamtx`
