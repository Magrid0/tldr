# pacfile

> Mostra informazioni sui file dei pacchetti, inclusi dati del repository, controlli del filesystem e confronti del database.
> Maggiori informazioni: <https://github.com/andrewgregory/pacutils/blob/master/doc/pacfile.pod>.

- Elenca tutti i pacchetti disponibili:

`pacfile {{percorso/del/file_o_directory}}`

- Confronta i valori del database con il filesystem:

`pacfile {{percorso/del/file_o_directory}} --check`

- Mostra aiuto:

`pacfile --help`

- Mostra versione:

`pacfile --version`
