# pridecat

> Come cat ma più colorato.
> Maggiori informazioni: <https://github.com/lunasorcery/pridecat#flags>.

- Mostra i contenuti di un file nei colori pride su `stdout`:

`pridecat {{percorso/del/file}}`

- Mostra i contenuti di un file nei colori trans:

`pridecat {{percorso/del/file}} {{[--trans|--transgender]}}`

- Alterna tra le bandiere pride lesbica e bisessuale:

`pridecat {{percorso/del/file}} --lesbian {{[--bi|--bisexual]}}`

- Mostra i contenuti di un file con i colori di sfondo cambiati:

`pridecat {{percorso/del/file}} {{[-b|--background]}}`

- Elenca i contenuti di una directory nei colori della bandiera pride:

`ls | pridecat --{{flag}}`
