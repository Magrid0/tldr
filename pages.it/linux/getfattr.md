# getfattr

> Mostra nomi di file e attributi estesi.
> Maggiori informazioni: <https://manned.org/getfattr>.

- Recupera tutti gli attributi estesi di un file e li mostra in formato dettagliato:

`getfattr {{[-d|--dump]}} {{path/to/file}}`

- Ottiene un attributo specifico di un file:

`getfattr {{[-n|--name]}} user.{{attribute_name}} {{path/to/file}}`
