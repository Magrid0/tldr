# trace-cmd list

> Mostra tracer, eventi o opzioni disponibili per il tracing.
> Vedi anche: `trace-cmd record`, `trace-cmd report`.
> Maggiori informazioni: <https://manned.org/trace-cmd-list>.

- Elenca i tracer disponibili:

`sudo trace-cmd list -t`

- Elenca i plugin disponibili:

`sudo trace-cmd list -p`

- Elenca gli eventi disponibili:

`sudo trace-cmd list -e`

- Elenca le funzioni disponibili:

`sudo trace-cmd list -f`
