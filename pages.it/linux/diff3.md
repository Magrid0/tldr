# diff3

> Confronta tre file riga per riga.
> Maggiori informazioni: <https://www.gnu.org/software/diffutils/manual/diffutils.html#Invoking-diff3>.

- Confronta file:

`diff3 {{path/to/file1}} {{path/to/file2}} {{path/to/file3}}`

- Mostra tutte le modifiche, evidenziando i conflitti:

`diff3 {{[-A|--show-all]}} {{path/to/file1}} {{path/to/file2}} {{path/to/file3}}`
