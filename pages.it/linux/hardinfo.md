# hardinfo

> Mostra informazioni hardware in finestra GUI.
> Maggiori informazioni: <https://manned.org/hardinfo>.

- Avvia `hardinfo`:

`hardinfo`

- Stampa il rapporto su `stdout`:

`hardinfo {{[-r|--generate-report]}}`

- Salva il rapporto in un file HTML:

`hardinfo {{[-r|--generate-report]}} {{[-f|--report-format]}} html > hardinfo.html`
