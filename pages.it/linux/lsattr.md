# lsattr

> Elenca gli attributi dei file su un filesystem Linux.
> Maggiori informazioni: <https://manned.org/lsattr>.

- Mostra gli attributi dei file nella directory corrente:

`lsattr`

- Elenca gli attributi dei file in un percorso specifico:

`lsattr {{path}}`

- Elenca gli attributi dei file ricorsivamente nella directory corrente e successive:

`lsattr -R`

- Mostra gli attributi di tutti i file nella directory corrente, inclusi quelli nascosti:

`lsattr -a`

- Mostra gli attributi delle directory nella directory corrente:

`lsattr -d`
