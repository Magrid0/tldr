# evince

> Visualizza documenti.
> Maggiori informazioni: <https://manned.org/evince>.

- Apre un file PDF:

`evince {{path/to/file.pdf}}`

- Apre più documenti:

`evince {{file1.pdf file2.pdf ...}}`

- Apre in modalità a schermo intero:

`evince {{[-f|--fullscreen]}} {{path/to/file.pdf}}`

- Apre in modalità presentazione:

`evince {{[-s|--presentation]}} {{path/to/file.pdf}}`

- Apre un numero di pagina specifico:

`evince {{[-i|--page-index]}} {{5}} {{path/to/file.pdf}}`
