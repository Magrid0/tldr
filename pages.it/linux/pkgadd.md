# pkgadd

> Aggiunge un pacchetto a un sistema CRUX.
> Maggiori informazioni: <https://docs.oracle.com/cd/E88353_01/html/E72487/pkgadd-8.html>.

- Installa un pacchetto software locale:

`pkgadd {{pacchetto}}`

- Aggiorna un pacchetto già installato da un pacchetto locale:

`pkgadd -u {{pacchetto}}`
