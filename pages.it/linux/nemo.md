# nemo

> Gestisce file e directory nell'ambiente desktop Cinnamon.
> Maggiori informazioni: <https://manned.org/nemo>.

- Apre la home directory dell'utente corrente:

`nemo`

- Apre directory specifiche in finestre separate:

`nemo {{percorso/della/directory1 percorso/della/directory2 ...}}`

- Apre directory specifiche in schede:

`nemo --tabs {{percorso/della/directory1 percorso/della/directory2 ...}}`

- Apre una directory con una dimensione specifica della finestra:

`nemo --geometry={{600}}x{{400}} {{percorso/della/directory}}`

- Chiude tutte le finestre:

`nemo --quit`
