# engrampa

> Comprime file in file zip/`.tar` nell'ambiente desktop MATE.
> Vedi anche: `zip`, `tar`.
> Maggiori informazioni: <https://github.com/mate-desktop/engrampa>.

- Avvia Engrampa:

`engrampa`

- Apre archivi specifici:

`engrampa {{path/to/archivio1.tar path/to/archivio2.tar ...}}`

- Comprime file e/o directory specifici ricorsivamente:

`engrampa --add-to={{path/to/compresso.tar}} {{path/to/file_o_directory1 path/to/file_o_directory2 ...}}`

- Estrae file e/o directory da archivi in un percorso specifico:

`engrampa --extract-to={{path/to/directory}} {{path/to/archivio1.tar path/to/archivio2.tar ...}}`
