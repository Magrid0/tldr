# x0vncserver

> Server TigerVNC per display X.
> Maggiori informazioni: <https://tigervnc.org/doc/x0vncserver.html>.

- Avvia un server VNC usando un file di password:

`x0vncserver -display {{:0}} -passwordfile {{path/to/file}}`

- Avvia un server VNC usando una porta specifica:

`x0vncserver -display {{:0}} -rfbport {{port}}`
