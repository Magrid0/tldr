# yetris

> Clone del gioco Tetris nel terminale.
> Maggiori informazioni: <https://github.com/alexdantas/yetris#controls>.

- Avvia una partita a Tetris:

`yetris`

- Muovi il pezzo orizzontalmente:

`{{<ArrowLeft>|<ArrowRight>}}`

- Ruota il pezzo in senso orario o antiorario:

`{{<x>|<z>}}`

- Tieni un pezzo (solo uno consentito alla volta):

`<c>`

- Soft drop del pezzo:

`<ArrowDown>`

- Hard drop del pezzo:

`<Space>`

- Metti in pausa/riprendi la partita:

`<p>`

- Esci dal gioco:

`<q>`
