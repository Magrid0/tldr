# do-release-upgrade

> Lo strumento di aggiornamento delle release di Ubuntu.
> Maggiori informazioni: <https://manned.org/do-release-upgrade>.

- Aggiorna all'ultima release:

`sudo do-release-upgrade`

- Aggiorna all'ultima release di sviluppo:

`sudo do-release-upgrade {{[-d|--devel-release]}}`

- Aggiorna all'ultima release proposta:

`sudo do-release-upgrade {{[-p|--proposed]}}`
