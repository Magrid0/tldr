# sensible-editor

> Apre l'editor predefinito.
> Maggiori informazioni: <https://manned.org/sensible-editor>.

- Apre un file nell'editor predefinito:

`sensible-editor {{percorso/del/file}}`

- Apre un file nell'editor predefinito, con il cursore alla fine del file:

`sensible-editor + {{percorso/del/file}}`

- Apre un file nell'editor predefinito, con il cursore all'inizio della riga 10:

`sensible-editor +10 {{percorso/del/file}}`

- Apre 3 file in finestre dell'editor divise verticalmente contemporaneamente:

`sensible-editor -O3 {{percorso/del/file1 percorso/del/file2 percorso/del/file3}}`
