# explodepkg

> Estrae il contenuto di un pacchetto Slackware nella directory corrente.
> Vedi anche: `installpkg`, `removepkg`, `upgradepkg`, `makepkg`, `pkgtool`.
> Maggiori informazioni: <https://www.slackbook.org/html/book.html#PACKAGE-MANAGEMENT-EXPLODEPKG>.

- Estrae un pacchetto nella directory corrente:

`explodepkg {{path/to/pacchetto.tgz}}`

- Estrae più pacchetti nella directory corrente:

`explodepkg {{path/to/pacchetto1.tgz}} {{path/to/pacchetto2.tgz}}`
