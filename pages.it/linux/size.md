# size

> Mostra le dimensioni delle sezioni all'interno di file binari.
> Maggiori informazioni: <https://sourceware.org/binutils/docs/binutils/size.html>.

- Mostra la dimensione delle sezioni in un dato file oggetto o eseguibile:

`size {{percorso/del/file}}`

- Mostra la dimensione delle sezioni in un dato file oggetto o eseguibile in [o]ttale:

`size {{[-o|--radix=8]}} {{percorso/del/file}}`

- Mostra la dimensione delle sezioni in un dato file oggetto o eseguibile in [d]ecimale:

`size {{[-d|--radix=10]}} {{percorso/del/file}}`

- Mostra la dimensione delle sezioni in un dato file oggetto o eseguibile in e[s]adecimale:

`size {{[-x|--radix=16]}} {{percorso/del/file}}`
