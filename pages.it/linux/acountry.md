# acountry

> Stampa il paese in cui si trova un indirizzo IPv4 o un nome host.
> Maggiori informazioni: <https://manned.org/acountry>.

- Stampa il paese in cui si trova un indirizzo IPv4 o un host:

`acountry {{example.com}}`

- Stampa output di [d]ebug aggiuntivo:

`acountry -d {{example.com}}`

- Stampa informazioni più [v]erbose:

`acountry -v {{example.com}}`
