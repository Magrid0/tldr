# fbi

> Mostra immagini in un terminale framebuffer.
> Maggiori informazioni: <https://manned.org/fbi>.

- Mostra un'immagine:

`sudo fbi {{path/to/file}}`

- Mostra più immagini:

`sudo fbi {{path/to/file1 path/to/file2 ...}}`

- Avvia una presentazione con un ritardo personalizzato (in secondi):

`sudo fbi {{[-t|--timeout]}} {{delay}} {{path/to/files/*}}`

- Adatta l'immagine per adattarla allo schermo:

`sudo fbi {{[-a|--autozoom]}} {{path/to/file}}`

- Mostra aiuto:

`fbi {{[-h|--help]}}`
