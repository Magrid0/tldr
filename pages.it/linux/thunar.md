# thunar

> Gestore di file grafico per ambienti desktop XFCE.
> Vedi anche: `caja`, `dolphin`, `nautilus`, `mc`.
> Maggiori informazioni: <https://docs.xfce.org/xfce/thunar/start>.

- Apri una nuova finestra che mostra la directory corrente:

`thunar`

- Apri l'utility di rinomina in blocco:

`thunar --bulk-rename`

- Chiudi tutte le finestre di thunar aperte:

`thunar --quit`
