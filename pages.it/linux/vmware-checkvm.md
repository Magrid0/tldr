# vmware-checkvm

> Verifica se l'host corrente è una VM VMware o meno.
> Maggiori informazioni: <https://manned.org/vmware-checkvm>.

- Restituisce la versione corrente del software VMware (lo stato di uscita determina se il sistema è una VM o meno):

`vmware-checkvm`

- Restituisce la versione hardware VMware:

`vmware-checkvm -h`
