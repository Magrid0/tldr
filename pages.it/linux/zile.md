# zile

> Un clone leggero dell'editor di testo Emacs.
> Maggiori informazioni: <https://manned.org/zile>.

- Avvia un buffer per note temporanee, che non verranno salvate:

`zile`

- Apri un file:

`zile {{path/to/file}}`

- Salva un file:

`<Ctrl x><Ctrl s>`

- Esci:

`<Ctrl x><Ctrl c>`

- Apri un file a un numero di riga specificato:

`zile +{{numero_riga}} {{path/to/file}}`

- Annulla le modifiche:

`<Ctrl x><u>`
