# slurmdbd

> Un'interfaccia enterprise sicura a un database per Slurm.
> Maggiori informazioni: <https://slurm.schedmd.com/slurmdbd.html>.

- Imposta il nice value del demone al valore specificato, tipicamente un numero negativo:

`slurmdbd -n {{valore}}`

- Cambia la directory di lavoro di `slurmdbd` al percorso LogFile o a `/var/tmp`:

`slurmdbd -s`

- Mostra aiuto:

`slurmdbd -h`

- Mostra versione:

`slurmdbd -V`
