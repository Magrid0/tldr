# eopkg

> Gestore di pacchetti per Solus.
> Maggiori informazioni: <https://help.getsol.us/docs/user/package-management/basics/>.

- Installa un pacchetto specifico:

`sudo eopkg install {{pacchetto}}`

- Aggiorna tutti i pacchetti:

`sudo eopkg upgrade`

- Cerca pacchetti:

`sudo eopkg search {{termine_ricerca}}`
