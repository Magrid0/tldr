# pkgctl build

> Compila pacchetti all'interno di una `chroot` pulita.
> Maggiori informazioni: <https://manned.org/pkgctl-build>.

- Sceglie automaticamente lo script di compilazione giusto per compilare pacchetti in una `chroot` pulita:

`pkgctl build`

- Compila manualmente pacchetti in una `chroot` pulita:

`pkgctl build --arch {{architettura}} --repo {{repository}} --clean`
