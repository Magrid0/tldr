# inputattach

> Collega un dispositivo al sistema di input Linux.
> I dispositivi di input sono presentati come file in `/dev/input/`.
> Maggiori informazioni: <https://manned.org/inputattach>.

- Collega un dispositivo Pulse8 CEC al sistema di input:

`inputattach --pulse8-cec {{/dev/ttyACM0}}`

- Mostra aiuto:

`inputattach --help`
