# getent

> Ottiene voci dalle librerie Name Service Switch.
> Maggiori informazioni: <https://manned.org/getent>.

- Ottiene l'elenco di tutti i gruppi:

`getent group`

- Mostra i membri di un gruppo:

`getent group {{group_name}}`

- Ottiene l'elenco di tutti i servizi:

`getent services`

- Trova un nome utente tramite UID:

`getent passwd 1000`

- Esegue una ricerca DNS inversa:

`getent hosts {{host}}`
