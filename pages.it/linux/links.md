# links

> Browser web a riga di comando solo testo.
> Vedi anche: `links2`.
> Maggiori informazioni: <https://links.twibright.com/user_en.html#subch-browser_calibration>.

- Visita un sito web:

`links {{https://example.com}}`

- Applica restrizioni per account anonimo:

`links -anonymous {{https://example.com}}`

- Abilita i Cookie (`1` per abilitare):

`links -enable-cookies {{0|1}} {{https://example.com}}`

- Naviga avanti e indietro tra i collegamenti di una pagina:

`{{<ArrowUp>|<ArrowDown>}}`

- Vai avanti e indietro di una pagina:

`{{<ArrowLeft>|<ArrowRight>}}`

- Esce:

`<q><y>`
